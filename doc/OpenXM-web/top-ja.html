  • Debian パッケージ opexm (Debian Etch 用)は こちらから 入手できます. こちらに 注意事項が書いてありますのでご覧下さい. (apt-get update が使えないのでご注意) (Aug 28, 2007)
    問題点: (1) /bin/sh が /bin/bash の symbolic link でないとインストールスクリプトがエラーで止ります. (2) gcc がインストールしてない場合(etch の標準デスクトップ環境など) "apt-get install gcc" で gcc をインストールして下さい.

  • cfep 1.1.2 を公開 しました(March 23, 2006). cfep は MacOS X 10.4 用の OpenXM ユーザインタフェースです. デフォルトの計算エンジンはRisa/Asir です.
  • Asir-contrib に y_prime (素数判定等), fj_curve (特異点解消の dual graph など), nk_mora (tangent cone algorithm for D, 局所b関数) が加わりました (Nov 11, 2005).

OpenXM とは?

数学に関する通信をやるための基礎技術です. 数学に関する通信をするためのプロトコルを提案しています. (OpenXM-RFC's). 数学データの標準化プロジェクトである OpenMath も応援してます. OpenXM では実際にいろいろな数学ソフトウエアをつないでパッケージを作成する 実験をやり, そのパッケージを配布しています. パッケージの画面コピーは こちら をご覧下さい. 以下はOpenXM パッケージの紹介のページです. functions 1, functions 2, Efficient computation.
