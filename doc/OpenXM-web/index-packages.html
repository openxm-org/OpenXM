OpenXM Distribution by Packages

ALL THE PACKAGES HAVE NOT BEEN READY FOR 1.2.2. USE THE SOURCE DISTRIBUTION.

FreeBSD 4 (RELEASE_1_2_2) INSTALLATION

  • Become the super user and input as follows if your shell is csh: 
     
  set path=(/usr/local/jdk1.1.8/bin $path)
  rehash
  pkg_add -r https://www.math.kobe-u.ac.jp/pub/OpenXM/1.2.2/openxm-1.2.1.tgz
    The package will be installed under /usr/local/OpenXM.

    Linux RPM (redhat package manager) (RELEASE_1_2_2, i386) INSTALLATION

  • Become the super user and type in 
      rpm -ivvh https://www.math.kobe-u.ac.jp/pub/OpenXM/1.2.2/openxm-shared-1.2.1-test.noarch.rpm
  rpm -ivvh https://www.math.kobe-u.ac.jp/pub/OpenXM/1.2.2/openxm-binary-1.2.1-test.i386.rpm
    The package will be installed under /usr/local/OpenXM.
    NOTE: There are a lot of reports on troubles of shared libraries on redhat Linux. Installation of shared libraries on Redhat must be done by hand (See FAQ). If you cannot stand for it, we recommend you to generate binaries from the source of the head branch (https://www.math.kobe-u.ac.jp/OpenXM/Head/openxm-head.tar.gz). You may fail to compile the gnuplot, but other modules may be compiled successfully. If you have questions, please feel free to send us an E-mail.

    FreeBSD 4 (RELEASE_1_2_2) Download the package

  • openxm-1.2.2.tgz [ the OpenXM package (Shift-Click)].

    Linux RPM (redhat package manager) (RELEASE_1_2_2) Download the packages

  • Architecture Independent Files (Library Files, Java Executables, Documents) openxm-1.2.2-test.noarch.rpm [ openxm-shared for Linux (Shift-Click)].
  • Binaries for i386 (Intel Pentium etc) (for libc 6 = glibc2) openxm-1.2.2-test.i386.rpm [ Intel CPU (Shift-Click) ]. This distribution does not contain the support for Mathematica.

    Debian GNU Linux(Debian package manager) (RELEASE_1_2_2) Download the packages

  • Download the two files below and become the super user and type in 
      dpkg --install openxm-shared_1.2.2-1_all.deb
  dpkg --install openxm-binary_1.2.2-1_i386.deb
  • Architecture Independent Files (Library Files, Java Executables, Documents) [ openxm-shared for Debian/GNU Linux (Shift-Click)].
  • Binaries for i386 (Intel Pentium etc) (for libc 6 = glibc2) openxm-binary_1.2.2-1_i386.deb [ Intel CPU (Shift-Click) ]. This distribution does not contain the support for Mathematica.

    How to delete packages

  • FreeBSD 
       pkg_delete openxm-1.2.2
  • Linux RPM 
       rpm -e openxm-binary-1.2.2-test
   rpm -e openxm-shared-1.2.2-test
  • Debian GNU Linux 
       dpkg --remove openxm-binary
   dpkg --remove openxm-shared
